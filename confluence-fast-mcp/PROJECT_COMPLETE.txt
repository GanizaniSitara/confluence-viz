================================================================================
  QUICK START
================================================================================

1. Install Dependencies:
   $ pip3 install --user -r requirements.txt

2. Configure (edit settings.ini):
   [data]
   pickle_dir = /home/user/git/confluence-viz/temp

3. Generate Pickles (if needed):
   $ cd /home/user/git/confluence-viz
   $ python3 sample_and_pickle_spaces.py

4. Run Tests:
   $ cd /home/user/git/confluence-fast-mcp
   $ python3 test_basic.py

5. Start Server:
   $ PYTHONPATH=$PWD/src python3 -m confluence_fast_mcp.server
